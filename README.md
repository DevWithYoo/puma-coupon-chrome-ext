# 🎰 룰렛 쿠폰 자동화 크롬 익스텐션

목표 할인율을 달성할 때까지 자동으로 룰렛을 돌려주는 크롬 확장 프로그램입니다.

## 설치 방법

1. 이 폴더를 다운로드하거나 클론합니다.

2. Chrome 브라우저에서 `chrome://extensions` 주소로 이동합니다.

3. 우측 상단의 **개발자 모드**를 활성화합니다.

4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다.

5. `roulette-coupon-extension` 폴더를 선택합니다.

6. 익스텐션이 설치되면 브라우저 우측 상단에 아이콘이 나타납니다.

## 사용 방법

1. 룰렛 이벤트가 있는 웹페이지로 이동합니다.

2. 브라우저 우측 상단의 익스텐션 아이콘을 클릭합니다.

3. **목표 할인율**을 입력합니다 (예: 30%).

4. **시작** 버튼을 클릭합니다.

5. 익스텐션이 자동으로:
   - 스토리지를 클리어하고 페이지를 새로고침합니다
   - '100% 당첨! 룰렛 돌리기' 버튼을 찾아 클릭합니다
   - 4초 후 획득한 쿠폰의 할인율을 확인합니다
   - 목표에 미달하면 위 과정을 반복합니다
   - 목표를 달성하면 알림을 표시하고 중지합니다

6. 수동으로 중지하려면 **중지** 버튼을 클릭합니다.

## 주의사항

- 이 익스텐션은 특정 웹사이트의 룰렛 이벤트에 맞게 제작되었습니다.
- '100% 당첨! 룰렛 돌리기'라는 텍스트와 'X% 쿠폰' 형식의 결과를 찾습니다.
- 다른 형식의 룰렛 이벤트에서는 동작하지 않을 수 있습니다.
- 과도한 사용은 해당 웹사이트의 이용약관에 위배될 수 있으니 주의하세요.

## 기술 스펙

- **Manifest Version**: V3
- **권한**: storage, activeTab, scripting
- **동작 방식**: Content Script + Background Service Worker
